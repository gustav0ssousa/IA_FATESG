## 📝 Sobre o Projeto

Este projeto tem como objetivo a ingestão, o processamento e a estruturação de dados em um banco de dados relacional **PostgreSQL**. O foco do trabalho é o tratamento do conjunto de dados `_Historical_Pandemic_Epidemic_Dataset_csv__202603252058.csv`, que oferece um registro histórico abrangente de eventos epidemiológicos significativos.

A base de dados abrange desde a Antiguidade (como a Peste Antonina) até a Era Contemporânea (como a COVID-19 e surtos de Sarampo). O objetivo principal da estruturação desses dados é permitir:

* **Análise Comparativa:** Avaliação entre diferentes patógenos (vírus, bactérias) e seus métodos de transmissão.
* **Avaliação de Contenção:** Estudo da eficácia de medidas de contenção ao longo dos séculos.
* **Estudos Estatísticos:** Facilitação de consultas analíticas sobre taxas de fatalidade, impacto econômico e a evolução dos avanços médicos em resposta a crises sanitárias globais e regionais.

## 🚀 Tecnologias Utilizadas

* **Linguagem Principal:** Python
* **Banco de Dados:** PostgreSQL
* **Manipulação de Dados:** `pandas`
* **Conexão e ORM:** `psycopg2`, `SQLAlchemy`
