# 🎓 Portfólio de Graduação - Inteligência Artificial (SENAI FATESG)

Bem-vindo ao meu repositório de estudos e práticas aplicadas em Inteligência Artificial. Este espaço é dedicado à consolidação dos conhecimentos adquiridos na graduação tecnológica do SENAI FATESG, com um foco profundo em como os dados são gerados, armazenados e transformados em valor estratégico.

## 📚 Escopo de Conhecimento

Os materiais, scripts e documentações organizados aqui refletem a base de dados e conectividade da minha formação, centralizados nos seguintes pilares:

* **Internet das Coisas (IoT):** Integração de sensores e hardwares, coleta de dados em tempo real e fundamentos de conectividade para criar a ponte entre o ambiente físico e as soluções de IA.
* **Business Intelligence (BI):** Análise de cenários de mercado, aplicação de estatística para negócios e definição de KPIs estratégicos para orientar a tomada de decisão baseada em dados.
* **Big Data:** Arquiteturas e pipelines focados no processamento e tratamento de grandes volumes de dados, preparando o terreno de forma otimizada para algoritmos de Machine Learning.
* **Bancos de Dados NoSQL:** Gerenciamento, modelagem e armazenamento de dados não relacionais, garantindo a flexibilidade e a escalabilidade exigidas por aplicações modernas.
